# Beauty Store App

This directory contains the PHP storefront and admin dashboard for Beauty Store.

For full setup, database, and usage instructions, see the repository README at `../README.md`.

Quick local URLs when the repository is placed under a web root as `ecom_proj`:

- Storefront: `http://localhost/ecom_proj/beauty-store/`
- Admin: `http://localhost/ecom_proj/beauty-store/admin/`
- Installer: `http://localhost/ecom_proj/beauty-store/install.php`

Default admin login:

```text
Email: admin@beauty.com
Password: password
```
